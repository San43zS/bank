# API Docs (Swagger / OpenAPI)

The OpenAPI specification for this service is located at:

- `docs/openapi.yaml`

## How to view

- Swagger Editor: paste the contents of `docs/openapi.yaml` into the editor.
- Postman / Insomnia: import `docs/openapi.yaml` as an OpenAPI definition.

## Auth

Protected endpoints use:

`Authorization: Bearer <access_token>`
